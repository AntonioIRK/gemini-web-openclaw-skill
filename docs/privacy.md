# Privacy and diagnostics

Diagnostics are useful for brittle browser automation, but they can contain private or account-adjacent data.

Possible artifacts include:

- screenshots
- HTML snapshots
- page text excerpts
- console output
- page URLs

## Rules

- treat diagnostics as private
- review before sharing
- do not commit diagnostics into git
- clean up artifacts when no longer needed

For the full guidance, see `../references/privacy-and-diagnostics.md`.
