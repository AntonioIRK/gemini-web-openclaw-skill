# Install

## Recommended path

```bash
cd ~/.openclaw/workspace/skills
git clone https://github.com/AntonioIRK/gemini-web-openclaw-skill.git gemini-web
cd gemini-web
./scripts/setup.sh
```

Then continue with:

- `./scripts/gemini_web_login.sh`
- `./scripts/gemini_web_doctor.sh`
- `./scripts/gemini_web_inspect_home.sh`

## Notes

- Python 3.11+ is required.
- A real browser login path is required.
- Local desktop install is the best-supported path.

For deeper prerequisites, see `../references/prerequisites.md`.
