# Support

## Best fit

This repo is best for operators evaluating or running Gemini Web automation in a self-hosted OpenClaw environment.

## Before filing an issue

Please check:
- `README.md`
- `references/validation.md`
- `references/remote-install-and-auth.md`
- `references/privacy-and-diagnostics.md`
- `references/support-matrix.md`

## Good issue reports include

- host OS and Python version
- whether install is local or remote
- which command you ran
- expected result
- actual result
- whether Gemini login was already completed in the persistent profile
- sanitized error output
- whether diagnostics were generated

## Do not post publicly

Do not include:
- browser profile contents
- cookies or session data
- screenshots with personal account data unless redacted
- full diagnostics bundles unless sanitized
- private prompts or user data unless necessary and redacted

## Security issues

For vulnerability disclosure, follow `SECURITY.md` instead of opening a public issue.
